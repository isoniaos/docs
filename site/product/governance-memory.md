# Governance Memory

Governance memory is the durable record of what an organization decided, why it decided it, what happened next, and what it learned.

## Memory should preserve

- proposal text and metadata;
- route and review context;
- discussion links;
- decision records;
- execution proofs;
- accountability updates;
- failures and cancellations;
- completion reviews;
- related proposals;
- policy changes and precedent.

## Public archive

The public governance archive should make past decisions inspectable without hiding uncertainty.

Useful filters include proposal type, status, responsible party, due date, execution state, and evidence source.

## Learning loop

Memory helps organizations avoid repeating unresolved decisions, missing follow-up obligations, and losing context across contributors, delegates, councils, and working groups.
