# Getting Started

The current public path is local development and evaluation. Exact commands live in the owning repository READMEs because each repository controls its own tooling, configuration, and validation.

## What You Can Run Locally

- EVM contracts on a local Hardhat chain.
- Control Plane with PostgreSQL, event indexing, projections, REST APIs, and diagnostics.
- App Core as a local React/Vite governance console.
- SDK, shared types, and theme packages for local package development.

## Maturity Boundary

These docs describe a developer-preview path. They do not present the local stack as production operations, a managed hosted service, or a complete provider-integration product.
