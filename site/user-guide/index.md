# User Guide

This guide explains how a non-technical user should think about using IsoniaOS.

The current product is still in developer-preview work, so exact screen names and button placement can change. The user model below is the intended public product flow.

## What You Can Do With IsoniaOS

You can use IsoniaOS to:

- create an organization record;
- choose a governance template;
- define roles and permissions;
- activate required setup;
- create proposal records;
- review and decide proposals;
- connect decisions to execution evidence;
- assign responsible people and due dates;
- track blocked, failed, cancelled, or complete follow-up;
- preserve a governance memory.

## Example Path

Community Grants DAO uses IsoniaOS like this:

1. Create an organization.
2. Choose a simple grants template.
3. Add reviewers, approvers, and executors.
4. Activate the organization after required settings are complete.
5. Approve a proposal to fund a contributor.
6. Attach transaction or external proof after the approved action.
7. Track the contributor's deliverable and completion review.
8. Preserve the final record for future participants.

Use the pages in this guide in that order if you are learning IsoniaOS for the first time.
