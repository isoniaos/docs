# Templates

Templates help an organization use a repeatable governance process.

## What A Template Does

A template can suggest:

- required proposal fields;
- reviewers;
- approvers;
- execution steps;
- evidence requirements;
- due-date rules;
- accountability statuses;
- archive structure.

## How To Choose A Template

Choose the template that matches the decision type.

| Need | Suggested template type |
| --- | --- |
| Fund contributor work | Grants or contributor funding. |
| Create a working group | Working-group mandate. |
| Change a technical setting | Technical change. |
| Review milestone delivery | Milestone review. |
| Record a simple decision | Basic proposal. |

Start simple. A template should help people understand the process, not make it harder to participate.

## Basic Flow

1. Open organization settings.
2. Review template options.
3. Choose the simplest matching template.
4. Review required fields and roles.
5. Edit names or guidance if the organization needs clearer wording.
6. Save the template.
7. Activate or re-activate settings if required.

## Example

Community Grants DAO chooses a grants template. The template asks every proposal to include a purpose, requested budget, responsible contributor, evidence expected after completion, and due date.

The organization can later add stricter review if the basic template is not enough.
